# scurvecalc

***

Calculate s-transformed marketing response

***

## Get it!

```
npm install --save scurvecalc
```

***

## Usage

``` js
var scurvecalc = require('scurvecalc')
var responseCurve = scurvecalc(803243, 7, 0.93939, 150000)
```

***

## API

### `scurvecalc(coeff, start, speed, N)`
